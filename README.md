# rn-logo-replacer
This is a project aiming at remplacing the logo of a react native app with a single npx command

## How to run it
### With basic
```
npm run change-icon
```
### All arguments
`--source <Path of a folder to copy>`   
`--destination <Path of a folder to copy to>` 
```
--operation Copy
            Move
            Delete
```
